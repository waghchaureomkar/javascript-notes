::first-line and ::first-letter

This is the first letter. This is an example of styling the first line and first letter of a paragraph. The first letter is larger and has a different color, while the first line is bold and has increased letter spacing.

5. CSS Variables (Custom Properties)

Defining and Using Variables

Primary Color
Secondary Color
Accent Color
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
}

.var-primary {
    background: var(--primary-color);
}

Variable with Fallback

Uses fallback value 
color: var(--undefined-color, #333);
/* If --undefined-color doesn't exist, use #333 */

Dynamic Theme Switching

Themed Content

Click buttons above to change theme

:root {
    --bg-color: white;
    --text-color: black;
}

[data-theme="dark"] {
    --bg-color: #333;
    --text-color: white;
}

.themed-content {
    background: var(--bg-color);
    color: var(--text-color);
}

Variables in Calculations

Variable in calc()
:root {
    --base-padding: 20px;
}

.box {
    padding: calc(var(--base-padding) * 2);
}

Responsive Font Sizes

This text uses calc() for responsive sizing: calc(16px + 1vw)

/* Grows with viewport width */

Complex Calculations

Complex calc example 
width: calc((100% - 60px) / 3);
padding: calc(20px * 2 + 10px);

Calc with CSS Variables

calc() with variables 
:root {
    --base-size: 100px;
    --multiplier: 2;
}

.box {
    width: calc(var(--base-size) * var(--multiplier));
}
