7. Semantic HTML5 Elements

Why Semantic HTML?

Semantic HTML provides meaning to web content, improving:

  • SEO (Search Engine Optimization)
  • Accessibility (Screen readers)
  • Code Readability
  • Maintainability

Semantic Elements

<header>
Header content, navigation, logo
<nav>
Navigation links
<main>
Main content (only one per page)
<section>
Thematic grouping of content
<article>
Independent, self-contained content
<aside>
Sidebar, related content
<footer>
Footer content, copyright, links
<figure> & <figcaption>
Images with captions
<details> & <summary>
Collapsible content
<time>
Date and time
<mark>
Highlighted text

Non-Semantic vs Semantic

❌ Non-Semantic:
<div class="header">...</div>
<div class="nav">...</div>
<div class="content">...</div>

✅ Semantic:
<header>...</header>
<nav>...</nav>
<main>...</main>
